2. Experiência perfeita no telemóvel
Os problemas mais comuns em mobile são:

Textarea muito pequeno — difícil de escrever

Botões pequenos demais — difícil de clicar

Nav horizontal com overflow — corta elementos

Font size < 16px — iOS faz zoom automático ao focar input (irritante!)

Height: 100vh — barra do browser iOS come espaço em baixo

Fix crítico iOS — o textarea precisa de fontSize: 16 mínimo para evitar zoom:

js
fontSize: 16,  // ← nunca menos que 16 em mobile!
Fix altura iOS — substitui 100vh por:

js
height: "100dvh"  // dynamic viewport height — respeita barra do Safari
